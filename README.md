# Template Astro Skeleton Svelte

A responsive GitHub Page built with [Astro](https://astro.build/) and [Skeleton](https://www.skeleton.dev/), an adaptive design system powered by Tailwind CSS.

## 🚀 Live Demo

[View the website](https://mistahuman.github.io/template-astro-svelte-skeleton)

## 🧰 Tech Stack

- **Astro** – Static site builder
- **Svelte** – Framework used in Skeleton components
- **Skeleton** – UI components and utilities based on Tailwind CSS
- **Tailwind CSS** – Utility-first CSS framework

## 📦 Installation

To run the project locally:

```bash
git clone https://github.com/mistahuman/template-astro-svelte-skeleton.git
cd template-astro-svelte-skeleton
npm install
npm run dev
```
